\chapter{Aufgabenstellung}
Das zu entwickelnde System ist eine Komponente, die einem Busfahrer Informationen über die Busse einer Buslinie zur Verfügung stellen soll. Dabei soll das System nicht nur auf Busse beschränkt sein. Denkbar sind hier auch andere Arten von öffentlichen Transportmitteln. Dabei wird die Komponente den zeitlichen Abstand zum vorherigen und nachfolgenden Bus einer Linie visualisieren. Die Visualisierung geschieht durch eine Android Applikation.
